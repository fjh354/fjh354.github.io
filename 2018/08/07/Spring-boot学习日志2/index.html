对spring boot整合Mybatis的学习做一个总结

首先需要引入mybatis的配置：

调查了一下版本，将mybatis-spring-boot-starter的版本改为1.1.1或者1.3.0之后可能会报错，其中一点的原因是注解方式更换，https://blog.csdn.net/n447194252/article/details/77968737

引入Mysql驱动：


其中：version可去，估摸着不写version会自动使用默认版本的jar包，这里JDK1.7用不了mysql 6.0.6的包，会报错

在applications.properties配置文件中添加MySQL的配置
spring.datasource.driverClassName = com.mysql.jdbc.Driver
spring.datasource.url = jdbc:mysql://localhost:3306/testuseUnicode=true&characterEncoding=utf-8
spring.datasource.username = root
spring.datasource.password = root
Spring boot会自动加载spring.datasource.*相关配置，数据源就会自动注入到sqlSessionFactory中，sqlSessionFactory会自动注入到Mapper中

启动器加上注解@MapperScan(“Dao包路径”)

mapper两种写法：
1.正常interface配合xml
2.interface内直接写sql
详细参考：http://www.ityouknow.com/springboot/2016/11/06/spring-boo-mybatis.html
